# FastAPI Tutorial

FastAPI is a modern, fast (high-performance), web framework for building APIs with Python 3.7+.

Work through this tutorial by Brad Traversy to create a simple FastAPI application with middleware that measures request processing time.
https://youtu.be/8TMQcRcBnW8?si=6Hx4nAPxpdUTMc0V

# Starting the app

```
 uv run fastapi dev main.py
```
